# 🏗️ MasterCode: The Future of Tech Education

![MasterCode Banner](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?auto=format&fit=crop&q=80&w=1200)

> **"Free your mind... and the code will follow."**

MasterCode is not just another learning platform. It is a **High-Performance Cognitive Environment** designed for developers who want to break out of the "tutorial hell" and master architecture, clean code, and advanced systems like Micro Frontends.

---

## 💼 Pitch Comercial: A Dor que Resolvemos

As empresas hoje enfrentam três grandes gargalos:
1. **Time-to-Market Lento**: Arquiteturas monolíticas e código legado impedem a inovação rápida.
2. **Gap de Senioridade**: Desenvolvedores sabem a sintaxe, mas não entendem a **arquitetura** necessária para escalar.
3. **Inconsistência Visual**: Falta de Design Systems que funcionem em múltiplos frameworks.

**A MasterCode surgiu para criar os 1% do mercado.** Educamos desenvolvedores na arte de criar sistemas resilientes, escaláveis e agnósticos, utilizando o que há de mais moderno na engenharia de software global.

---

## 🚀 Tecnologias de Elite

O ecossistema MasterCode é construído sobre pilares de alta performance:

- **Frontend**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) para máxima tipagem e segurança.
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) para design atômico e ultra-rápido.
- **Backend & Real-time**: [Firebase](https://firebase.google.com/) (Auth & Cloud Firestore) para persistência e sincronização instantânea.
- **AI Intelligence**: Integração nativa com [Gemini AI](https://deepmind.google/technologies/gemini/) para monitoria inteligente.
- **Animations**: [Framer Motion](https://www.framer.com/motion/) para uma experiência de usuário (UX) fluida e imersiva.
- **Architecture**: Micro Frontends, Web Components e Design Systems Atômicos.
- **Charts**: [Recharts](https://recharts.org/) para visualização de progresso e XP.

---

## ✨ Funcionalidades Principais

- **Dual-Mode Theme**: Escolha entre a precisão da **Matrix (Code-focused)** ou a potência do **Ironman (Performance-driven)**.
- **Smart Blog/Newsletter**: Conteúdo técnico renderizado em Markdown com suporte a Podcasts e Vídeo Aulas integradas.
- **Comunidade Ativa**: Feed de discussões centralizado com sincronização em tempo real via Firestore.
- **XP & Gamificação**: Sistema de progressão de carreira integrado ao perfil do usuário.
- **AI Mentoring**: Assistente inteligente para tirar dúvidas sobre arquitetura e código.

## 🛠️ Como Executar o Projeto

### Pré-requisitos
- Node.js 18+
- Conta no Firebase (Configurações no `firebase-applet-config.json`)
- Chave de API do Gemini (Configurada no `.env`)

### Instalação
```bash
# Instalar dependências
npm install

# Iniciar em modo de desenvolvimento
npm run dev
```

### Build para Produção
```bash
npm run build
npm start
```
